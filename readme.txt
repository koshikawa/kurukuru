●操作方法

	右へ移動：→　または　６（テンキー）
	左へ移動：←　または　４（テンキー）
	落とす　：↓　または　２（テンキー）
	左回転　：Ｚ　または　Ｃ
	右回転　：Ｘ　または　５（テンキー）

●実行に必要なもの

	　クルクルコッシーを実行するためには、Visual Basic 5.0のランタイムライブ
	ラリが必要です。ランタイムライブラリがなにかわからないという方は、普通の
	パソコンには組み込まれているので安心してください。ではなぜ書くのかって、
	普通でないパソコンを使っていない方がいるからです。

●遊び方

	　クルクルコッシーの遊び方はいたって簡単。同じ色のクルクルコッシー（顔マー
	ク、時々コッシーと略します）を上下左右に3個以上つなげて消していくというも
	のです。赤い線より上にコッシーを積んでしまうと、ゲームオーバーです。

	　コッシーは２個セットで上から落ちてきます。それを回したり、移動させたりし
	て好きなところに置いてください。その時、次に降ってくるセットの色が右上の位
	置に出ます。

	  たまに、小さなコッシーが出てきます。このコッシーはちびコッシーといって、
	そのままでは消すことができません。普通のコッシーを消すときにとなりにいる
	と、ちびコッシーは普通のコッシーに変わり、消すことができるようになります。

	　操作するコッシーの両側が壁やコッシーに挟まれているときでも、回転ボタン
	を素早く2回押すと回転することができます。

●ルール

	　素晴らしいことに、ルールを変化させることができます。メニューのルールから
	選んでください。これを変更することによって、256色消しモードにしたり、2個消
	しモードにしたりができます。初期状態に戻すことができますので、適当に変えて
	遊んでください。ゲームプレイ中は選ぶことができません。

	　フィールドの広さ…フィールドの広さを選ぶことができます。
	　色の数…色の数を設定します。普通は４です。多いと難しくなります。
	　つなげる数…同じ色のコッシーをいくつつなげると消えるかを設定します。
	　ななめで…つなげるにすると消えすぎて面白くなくなります。
	　まがって…つなげる途中で曲げても消えるかを決めます。
	　ちびコッシーの出現頻度…ちびコッシーを出なくしたり出したりします。

●オプション

	　ルールだけでなく他にもいろいろと変えることができます。メニューのオプショ
	ンから選んでください。ゲームプレイ中も選ぶことができます。

	　コッシーの降る速さ…コッシーが降ってくる速さを変更します。
	　下キーを押したとき…下キーを押したときの落ちかたを変えます。
	　効果音…効果音を鳴らすか鳴らさないかを決めます。
	　NEXTコッシーのプレフェッチ…NEXTコッシーを何組表示するか決めます。

●スコアの計算法

	　消したコッシーの数の100倍のスコアが入ります。連鎖で消すと、2連鎖目の点数
	は消した数の200倍、3連鎖目は400倍、4連鎖目は800倍、5連鎖目は1600倍と、倍倍
	に増えていきます。高得点を狙うには連鎖で消すのがコツです。

●効果音

	　クルクルコッシーは実は隠し機能として効果音が鳴らせるようになっています。
	オプションで選べる効果音をONにすることが効果音を鳴らすための初めの一歩で
	す。ただしそれだけでは効果音は鳴りません。kurukuru.exeと同じところにある
	kurukuru.wavというファイルに音が入っていないからです。このkurukuru.wavとい
	うファイルをサウンドレコーダーなどで音を録音することで、効果音が鳴らすため
	の準備ができました。あとは、ゲームを始めてコッシーをつなげるだけです。音は
	鳴りましたか。わからないというかたは、ここに書いてある説明をあなたのまわり
	のWindowsに詳しい人に読ませて、手伝ってもらってください。

●著作権

	　クルクルコッシーの著作権は越川康則にあります。
	　クルクルコッシーはフリーソフトです。
	　再配布は金銭が関与しない場合に限り自由です。金銭が関与する場合は事前に著
	作権者に連絡して、許可を取ってください。
